# BusynessNotification
これは起動後に遅いPCに対して、快適な動作をする状態になったことを通知するアプリケーションです。
HDDをシステムドライブにしたPCやスタートアップアプリが多いPCに対して最も効果的に働きます。
また、このアプリを常駐させることで、動作が重くなった際にどのリソースを開放するべきかを通知してくれる機能を追加予定です。
